Industry Type Application Overview Concepts used or deployed Lessons learned Refernce documents Reasearch or Deployed
Recieveing & Loadout Facility This facility would recieve, store, dry and then load out product. We used a CMC web server integrated into a Allen bradley PLC that made use of a Redlion as a gateway to translate the modbus TCP protocol to the AB PLC. At another similar facility we removed the red lion and talked directly to the PLC using a rockwell modbus TCP block. At another plant we used kepware to interface M340 with a Rockwell SE application. Depending on the product a lot of times sites willl tend to want to stop the conveyors loaded and not clean out. This minimizes downtime to get product moving again. Deployed
HVAC System for a Chemical Plant This facility would produce chemicals. We were in charge of keeping everything cool, exhausting or containing chemicals in the event of a leak or keeping things from freezing. We used the modbus TCP block here to talk to an OEM controller to tell AC Units to start stop and read some status values. Deployed
Pet Food processing Facility This facility would process and color pet food and then run it through an extruder. We made use of PlantPAX controls along with a simple from scratch batching routine that we developed. The heat created can be to much for a workstation to handle had to add an additional cooler to the panel in order for the application to run smoothly. Deployed
Packaging Facility We had to quickly move and dose a system that would then package a container that needed high accuracy. We used multiple servoes here to calculate the weight needed and based off a static value does the system with in a reasonable tollerance. Deployed
